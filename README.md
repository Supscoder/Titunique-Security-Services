# Titunique Security Services Website

A professional, responsive multi-page website for Titunique Security Services.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Static Header**: Fixed navigation header with logo on all pages
- **Mobile Menu**: Hamburger dropdown menu for mobile devices
- **Page Transitions**: Smooth rolling circle spinner animation between pages
- **Professional Branding**: Green and silver color scheme matching the company logo
- **Multiple Pages**: Home, Services, About, and Contact pages

## Pages

1. **Home (index.html)** - Welcome page with key highlights
2. **Services (services.html)** - Complete list of all security services
3. **About (about.html)** - Company mission, vision, and values
4. **Contact (contact.html)** - Contact information and inquiry form

## Files

- `index.html` - Home page
- `services.html` - Services page
- `about.html` - About page
- `contact.html` - Contact page
- `style.css` - Main stylesheet with responsive design
- `script.js` - JavaScript for navigation and animations
- `assets/images/logo.jpeg` - Company logo


## Customization

To customize the website:
- Edit `style.css` to change colors, fonts, or layout
- Edit HTML files to update content
- Replace `assets/images/logo.jpeg` with your own logo
- Update contact information in `contact.html`
